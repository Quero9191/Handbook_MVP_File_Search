## Purpose
DevRel owns the developer experience: clear docs, responsive support, and healthy community relationships.

## Scope
- Applies to: All customer-facing documentation and partnerships
- Doesn't apply to: Internal engineering processes (see handbook/)

## DevRel Responsibilities

### Documentation
- Keep API docs up-to-date
- Write integration guides
- Maintain FAQ and troubleshooting

### Community
- Monitor GitHub issues and discussions
- Respond to support requests
- Host webinars and office hours

### Partnerships
- Build integrations with partner platforms
- Manage release notes and announcements
